# Executive Sales Dashboard 📊

## 📌 Project Overview
This project focuses on building an interactive and insights-driven Sales Dashboard using Power BI. The goal is to analyze overall sales performance, revenue trends, and key business metrics to help corporate stakeholders make data-driven decisions.

---

## 🛠️ Tech Stack & Skills Used
* **Data Source:** Microsoft Excel (Sales Transactions Data)
* **Data Cleaning & ETL:** Power Query (Handling missing values, data type formatting)
* **Data Modeling:** Star Schema (Establishing relationships between tables)
* **Analytics & Metrics:** DAX (Data Analysis Expressions) for Calculated Columns & Measures (YoY Growth, Total Revenue, Average Order Value)
* **Visualization:** Power BI (Donut charts, KPI cards, Filled Maps for regional sales)

---

## 💡 Key Business Insights Documented
* **Top Performing Product:** Headphone category generated the highest volume of sales transactions.
* **Regional Performance:** Identified high-revenue zones and expansion opportunities using interactive geospatial (map) analysis.
* **Sales Trends:** Detected specific seasonal peaks in sales, helping supply chain teams plan inventory optimization.

---

## 📂 Repository Structure
* `Executive-Sales-Dashboard.pbix` -> Power BI Dashboard File (Download and open in Power BI Desktop)
* `Sales_Data.xlsx` -> Raw Sales Dataset used for analysis
